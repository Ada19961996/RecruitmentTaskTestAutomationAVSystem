# RecruitmentTaskTestAutomationAVSystem

Test automation for recruitment app with Playwright + TypeScript

Make sure you have installed:

Node.js
npm or yarn

to check versions, input in command window:

node -v
npm -v

Project setup:

Create folder to store the project locally, open terminal in the folder and clone the repository
git clone https://github.com/Ada19961996/RecruitmentTaskTestAutomationAVSystem.git

Install dependencies:
npm install

Install Playwright:
npx playwright install

To run all tests:
npx playwright test

To run tests in UI mode:
npx playwright test --ui

To run a specific test file:
npx playwright test tests/example.spec.ts

To run tests in headed mode (with visible browser):
npx playwright test --headed

To open HTML report:
npx playwright show-report

Enjoy!
